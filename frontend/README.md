# 초등 고학년 생성형 AI 체험 프론트

`pearl-image`의 컴포넌트 분리 구조(헤더/입력/결과 패널)를 참고해 만든 교육용 이미지 생성 프론트입니다.

## 핵심 기능

- OpenAI 이미지 생성(`gpt-image-1`) 프론트 UI
- 2단계 안전 필터
  - 로컬 금칙어 검사
  - OpenAI `omni-moderation-latest` 검사
- 기준 이미지 대비 유사도(%) 표시
- 기본 기준 이미지 2종 + 사용자 업로드 기준 이미지 지원

## 실행

```bash
cd frontend
npm install
npm run dev
```

## 사용 방법

1. OpenAI API Key 입력
2. 기준 이미지 선택 또는 업로드
3. 캐릭터 프롬프트 작성
4. `안전 검사 후 이미지 생성` 버튼 클릭
5. 생성 결과와 유사도 확인

## 참고

- 브라우저에서 직접 OpenAI API를 호출하므로 수업/실험 환경에서만 사용하세요.
- 운영 환경에서는 API Key 보호를 위해 서버 프록시를 권장합니다.
